An accessible chess board web component.

A simple, but feature-rich chess board that allows moves via mouse and keyboard.

Easy to integrate.

Add a chess board to your page is as simple as adding an HTML tag to your document. And it has out-of-the-box support for mouse and keyboard moves.

Tip: press Tab to focus the board and use the keyboard to move around!
